Демонстрация встраивания чата

Как встроить чат на свой сайт

1. Добавьте конфигурацию чата перед подключением скрипта:

<script>
    window.CHAT_CONFIG = {
        siteId: 'your-site-id', // Уникальный идентификатор вашего сайта
        serverUrl: 'http://your-chat-server.com', // URL вашего сервера чата
        title: 'Ваш заголовок чата' // Заголовок виджета
    };
</script>

2. Подключите скрипт чата:

<script src="http://your-chat-server.com/js/chat-embed.js"></script>

Пример использования

Вот как это будет выглядеть на вашем сайте:

<!DOCTYPE html>
<html>
<head>
    <title>Ваш сайт</title>
</head>
<body>
    <!-- Ваш контент сайта -->

    <script>
        window.CHAT_CONFIG = {
            siteId: 'my-awesome-site',
            serverUrl: 'http://your-chat-server.com',
            title: 'Поддержка клиентов'
        };
    </script>
    <script src="http://your-chat-server.com/js/chat-embed.js"></script>
</body>
</html>
